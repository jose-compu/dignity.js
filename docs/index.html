dignity.js documentation

Version 0.4.0 - P2P object API for decentralized JavaScript applications

Quick start

const {
  DignityP2P,
  InMemoryNetworkHub,
  InMemoryNetworkAdapter
} = require('dignity.js');

const hub = new InMemoryNetworkHub();

const alice = new DignityP2P({
  nodeId: 'alice',
  networkAdapter: new InMemoryNetworkAdapter(hub),
  security: {
    appPassword: 'shared-out-of-band-password',
    powSteps: 22
  }
});

await alice.start();
await alice.create('notes', { title: 'hello world' }, { id: 'note-1' });
const note = alice.read('notes', 'note-1');

REST-like object API

  • create(collection, data, options) -> owner is the creator.
  • read(collection, id) -> returns object or null.
  • list(collection, options) -> list active objects (or include deleted markers).
  • update(collection, id, patch, options) -> only object owner can update.
  • updateWithRetry(collection, id, patchFn, options) -> optimistic read-modify-write retries.
  • remove(collection, id) -> only object owner can delete.

Optimistic concurrency

node.on('conflict', (event) => {
  console.log(event.phase, event.expectedVersion, event.currentVersion);
});

await node.update('games', 'g1', { score: 10 }, { expectedVersion: 3 });

Room / team discovery

await node.joinDiscovery('team:red', {\n  metadata: { nickname: 'alice' },\n  heartbeatIntervalMs: 15000,\n  ttlMs: 45000\n});\n\nconst peers = node.listPeers('team:red', { includeSelf: false });\nawait node.leaveDiscovery('team:red');

Security defaults (enabled)

  • Signing: every message is signed by sender keypair (Ed25519).
  • Broadcast encryption: AES secretbox with PBKDF2-SHA256 derived key from appPassword or scoped broadcastPasswords.
  • Direct encryption: NaCl box to recipient X25519 public key (true E2E for direct messages).
  • PoW: Sloth VDF proof per message, default powSteps: 22.
  • Peer bans: invalid signature or invalid PoW bans sender for 48h by default.
  • Legacy KDF: older single-hash broadcast messages remain decryptable.

IndexedDB persistence

const { IndexedDBPersistence } = require('dignity.js');

const persistence = new IndexedDBPersistence({
  dbName: 'my-app',
  collections: ['games']
});

await node.start();
await persistence.attach(node);

Signaling defaults and customization

const {
  createDefaultSignalingPool,
  WebSocketSignalingProvider
} = require('dignity.js');

const pool = createDefaultSignalingPool({
  cloudflareUrls: ['wss://your-cloudflare-endpoint.example'],
  fallbackUrls: ['wss://relay-a.example', 'wss://relay-b.example'],
  customProviders: [
    new WebSocketSignalingProvider({
      id: 'local-dev',
      url: 'ws://localhost:3001',
      priority: 99
    })
  ]
});

Machine-readable API metadata is available in openapi-like.json.
